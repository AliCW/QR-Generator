# QR Code Generator

Very simple FE resource that generates QR codes from the user input.

## Instructions

Clone the repo

    git clone https://github.com/AliCW/QR-Generator.git

Install dependencies

    npm install

Start the resource

    npm start

## Usage

Navigate to http://localhost:3000 - add text into input bar & click generate!
